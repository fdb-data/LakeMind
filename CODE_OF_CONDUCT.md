# 贡献者公约行为准则

## 我们的承诺

为了营造一个开放友好的环境，我们承诺为所有参与本项目的人提供不受骚扰的体验，无论其身份背景如何。

## 我们的准则

**鼓励的行为**：

- 使用友好包容的语言
- 尊重不同观点和经验
- 优雅地接受建设性批评
- 关注社区整体利益
- 对其他社区成员展现同理心

**不可接受的行为**：

- 使用性化语言或 imagery
- 人身攻击、侮辱性评论、恶意挑衅
- 公开或私下骚扰
- 未经许可泄露他人私人信息
- 其他违反职业操守的行为

## 执行

违反本准则的行为可由维护者删除、编辑或拒绝。维护者有权在认为行为不当时暂时或永久禁止贡献者参与社区。

## 适用范围

本行为准则适用于所有项目空间，包括代码仓库、Issue、Pull Request、讨论区、聊天室等。

## 联系

如遇违反行为准则的情况，请通过 [GitHub Issues](https://github.com/anomalyco/LakeMind/issues) 私下联系项目维护者。
